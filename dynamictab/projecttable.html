What I learned in this project:

In this project I learned how to node js and how to make a rest api in node js and use it to communicate with the database and also learned a new database management which is mongodb.

I also learned creating database and collections in mongodb using mongoose module of nodejs, creating dynamic tables, querying record, populating records, using $elemMatch to in queries sorting the result of query and getting data between two dates.I also learned CRUD operations on dynamically created tables.I also learned different ways of doing this operations using mongoose module of nodejs.

Besides this, in nodejs I learned how to authenticate user using passport register user and also restrict the operation of users and make them based on tokens.I learned insatlling packages in rest API both using totally commandline approach and using package.json and commandline.I also gained experience about the asynchronous nature of nodejs.I also learned how to use express routers to make the project modular.I learned how to get the visitor's headers and ip addres using nodejs and storing in the collection so as to make our content more user friendly depending on the visitors.

I also learned creating and managing repository in bitbucket using git, commiting changes and sending the commits to repository.
